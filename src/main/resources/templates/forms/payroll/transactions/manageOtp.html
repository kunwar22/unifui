  Manage OTP Values

OTP stands for One Time Positive Input. OTP elements in payroll are elements that are not calculated through formulas and the default value for each OTP element is 0. You can override the default value by passing values against each employee for the desired element, through this interface.

Steps to Upload OTP Values:

  • Download OTP upload template from here
  • Fill the template.
  • Upload the file on the server from billow link.
